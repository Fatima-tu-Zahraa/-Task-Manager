# Task Manager

A task management app built with React.js and Tailwind CSS.

## Live Demo
[View Live](https://task-manager-nu-five-70.vercel.app)

## Features
- Add and delete tasks
- Mark tasks as complete/incomplete
- Filter tasks — All, Active, Completed
- Stats — Total, Done, Pending count
- Data saved with Local Storage

## Tech Stack
- React.js
- Tailwind CSS
- Vite
- Local Storage

## Getting Started
```bash
npm install
npm run dev
```
